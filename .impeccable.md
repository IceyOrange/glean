# Glean Design Context

## Design Context

### Users
- 阅读爱好者 / 知识工作者：浏览网页时随手采集金句（quote）和自己的感想（thought），形成「卡片」
- 多语言用户（中文默认，支持 English / Français）
- 使用场景：阅读中通过划词浮层快速保存；popup 快速查看统计与最近条目；history 页沉浸式回顾、搜索、AI 洞察
- 情感目标：安静、专注、书卷气——像一个随身携带的摘抄本（commonplace book），而不是一个效率仪表盘

### Brand Personality
- 三个词：**温润、书卷气、克制**（warm, bookish, restrained）
- 参考气质：Readwise Reader 的摘录页、纸质杂志的书摘版式、图书馆目录卡
- 反参考： SaaS 仪表盘风、霓虹渐变、AI 味默认灰白模板、过度卡片化

### Aesthetic Direction
- **温润编辑感（warm editorial）**：纸张般的中性暖色 + 墨色文字 + 一枚「印章红」点缀
- 主题：浅色默认，支持深色模式（跟随系统或手动切换）
- 字体策略：衬线展示字体（Literata，用于引用文、标题、数字）+ 系统无衬线正文（app UI 常规文本）；content script 浮层受环境限制使用 Georgia / Songti SC 系统衬线栈
- 关键记忆点：引用文以「印刷书摘」的版式呈现（衬线、悬挂引号、宽松行距）

### Design Principles
1. **Paper, not gray**：中性色一律带暖色调（OKLCH chroma 0.005–0.015），禁止纯灰、纯黑 #000、纯白 #fff
2. **一处朱砂**：印章红（seal red）是唯一强点缀，仅用于关键动作与强调；统计图表使用墨/朱砂/赭石的和谐三色，禁止高饱和默认色
3. **衬线即身份**：quote、数字、wordmark 用衬线；功能性 UI 文本用系统 sans；字号层级对比 ≥ 1.25
4. **安静的交互**：hover 才显现的次级操作、150–300ms ease-out 过渡、仅 transform/opacity 动画、尊重 prefers-reduced-motion
5. **留白分级**：4pt 间距刻度，紧凑分组 + 慷慨分隔；标题上方留白大于下方；拒绝到处加卡片和边框
